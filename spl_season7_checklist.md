# SPL Participant Pre-Event Checklist

## Table of Contents
1. Setup & Environment  
2. API Skills & Technical Know-How  
3. Model Usage & Limits  
4. Bot Logic & Game Flow  
5. Honor Code  
6. Allowed & Not Allowed Tools  

---

## 1. Setup & Environment
1. Install S-Box (and LabVIEW Runtime if required)  
2. Ensure Soliton VPN access (needed for pre-SPL only)  
3. Confirm access to SPL organization on OpenAI platform  
4. Confirm access to your project and create your OpenAI API key  
5. Install Python or chosen language runtime  
6. Install required packages and preferred editors  
7. Verify LAN/Wi-Fi connectivity at office  

---

## 2. API Skills & Technical Know-How
1. Have sample code for calling OpenAI Responses API  
2. Have sample code for calling OpenAI Conversation API  
3. Know how to make async, non-blocking LLM calls  
4. Know how to manage multiple games asynchronously from one bot  
5. Know how to use WebSockets for sending/receiving messages  
6. Understand and use JSON formats correctly  
7. Know how to use structured prompts with variables  
8. Able to debug OpenAI API errors with logs and systematic steps  

---

## 3. Model Usage & Limits
1. Know accessible models and when to use each  
2. Understand rate limits (RPM, TPM)  
3. Know model token costs and budgeting  
4. Able to choose the right model using reasoning or LLM help  
5. Track usage, limits, and spending in the OpenAI dashboard  

---

## 4. Bot Logic & Game Flow
1. Know how to maintain full game history/state in memory  
2. Produce structured, formatted responses reliably  
3. Implement fallback logic for slow LLM responses  
4. Implement detailed logs for all bot actions  
5. Debug, trace, and recover gracefully from runtime issues  

---

## 5. Honor Code
1. Play fair — use the system creatively to learn and have fun.  
2. Use only the officially provided SPL models during the event.  
3. You may use web search, free generative AI tools, and open-source tools.  
4. You may not use paid tools or subscriptions (e.g., GitHub Copilot, ChatGPT Plus).  
5. Declare all tools in the Gen AI Tools Declaration sheet.  
6. Keep your API key private — do not share it with anyone.  
7. During pre-SPL, openly share ideas, learnings, and code snippets.  
8. Develop your own work — do not copy or reuse someone else’s code.  
9. Report any bugs, loopholes, or misuse to organizers immediately.  
10. Uphold integrity and the Soliton spirit throughout SPL.  
11. Respect all final decisions made by organizers.  

---

## 6. Allowed & Not Allowed Tools

### Allowed
1. Any free generative AI tools  
2. Any open-source libraries/utilities  
3. Web search for clues and reasoning  

### Not Allowed
1. Any paid generative AI tools (e.g., GitHub Copilot, paid ChatGPT)  
2. Any paid plugins/extensions  
3. Any tools requiring licensed access beyond SPL
